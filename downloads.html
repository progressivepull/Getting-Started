Software Downloads



Git a free and open source distributed version control system designed to handle everything from small to very large projects with speed and efficiency.

SourceTree simplifies how you interact with your Git repositories so you can focus on coding. Visualize and manage your repositories through SourceTrees simple Git GUI.

Node.js is a free, open-source, cross-platform JavaScript runtime environment that lets developers create servers, web apps, command line tools and scripts. Node.js comes with npm, the standard package manager for Node.js.

GitHub Desktop is a free and open-source application designed to simplify your development workflow. It allows you to interact with GitHub repositories using a graphical user interface (GUI) instead of the command line or web browser.

Visual Studio Code, often referred to as VS Code, is a free, lightweight, yet powerful source-code editor developed by Microsoft.

Anaconda is a popular distribution of Python and R designed for data science, machine learning, and scientific computing. It comes with a package manager, environments for dependency management, and a collection of pre-installed libraries like NumPy, pandas, and TensorFlow. Anaconda simplifies package installation, version control, and workflow management, making it a go-to tool for professionals and researchers working with data.
